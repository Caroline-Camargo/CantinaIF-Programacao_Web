O QUE É O CANTINAIF:


O CantinaIF é um site realizado para a utilização na Cantina do IFSul Câmpus Camaquã. Dentre os principais recursos estão a disponibilização de um cardapio dos lanches disponíveis e uma página de login para que os usuários possam ver suas dívidas pendentes com a Cantina. O site visa ter uma relação mais próxima com os alunos do Câmpus e utilizar a tecnologia para oferecer um serviço mais completo para os estudantes, uma vez que a lista de lanches pode ser acessada de qualquer lugar. Foi desenvolvido em uma linguagem de marcação HTML e para a estilização foi utilizado o CSS. O mesmo foi desenvolvido em sala de aula como parte da avaliação da disciplina de programação do quarto ano do curso Técnico em informática no Instituto Federal Sul Riograndense Câmpus Camaquã.
